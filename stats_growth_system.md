# 능력치 & 육성 시스템

## 📊 8개 핵심 능력치 시스템

### 능력치 구성 (Football Manager 스타일)
```
전투 (Combat)
- 범위: 1-25
- 영향: 근접 전투 효율, 몬스터와의 교전 성과
- 활용: 방어전, 탐사 중 전투, 경비 임무

사격 (Shooting) 
- 범위: 1-25
- 영향: 원거리 무기 정확도, 사격 피해량
- 활용: 감시탑 운용, 원거리 전투, 사냥

체력 (Physical)
- 범위: 1-25  
- 영향: 생존력, 부상 저항, 지구력
- 활용: 모든 활동의 기본, 생존 확률

민첩 (Agility)
- 범위: 1-25
- 영향: 이동속도, 회피율, 반응속도  
- 활용: 탐사, 정찰, 위험 회피

작업 (Working)
- 범위: 1-25
- 영향: 건설 효율, 시설 운영, 제작 품질
- 활용: 건설 작업, 시설 유지보수, 생산 활동

기술 (Technical)
- 범위: 1-25
- 영향: 전문 기술 활용, 복잡한 장비 조작
- 활용: 연구, 고급 시설 운영, 장비 수리

의료 (Medical)
- 범위: 1-25
- 영향: 치료 성공률, 바이러스 대응, 의학 연구
- 활용: 부상 치료, 질병 예방, 의료 연구

정신 (Mental)
- 범위: 1-25
- 영향: 스트레스 저항, 판단력, 리더십
- 활용: 사기 관리, 의사결정, 위기 상황 대처
```

## ⭐ 특성 시스템

### 특성 유형 분류

#### 🔥 전투 특성
```
근접 전문가
- 효과: 근접 전투 피해 +25%
- 조건: 전투 15+ 또는 특별 획득

사격 명수
- 효과: 원거리 정확도 +30%, 크리티컬 확률 +10%
- 조건: 사격 15+ 또는 특별 획득

방어 전문가  
- 효과: 받는 피해 -20%, 방어 시설 효율 +15%
- 조건: 체력 15+ 또는 특별 획득

야간 작전
- 효과: 야간 활동 효율 +50%, 야간 전투 보너스
- 조건: 민첩 12+ 또는 특별 획득
```

```
효율적 작업

장비 전문가
```

```
위생 관리자
```

### 특성 획득 방법

#### 태생 획득
```
캐릭터 생성시 보유:
- 초기 생존자: 0-1개 특성 (개인 배경에 따라)
- 소환 캐릭터: 등급에 따라 1-3개 특성
- 고유한 개성과 전문성을 나타내는 핵심 요소
```

#### 아이템 사용 (고가치 희귀 아이템)
```
전투 교범 (극희귀)
- 효과: 전투 관련 특성 1개 습득
- 획득처: 군사 기지 최심층부, 특수 임무 보상
- 존재량: 게임당 2-3개 제한

의학 서적 (극희귀)  
- 효과: 의료 관련 특성 1개 습득
- 획득처: 대학병원 연구실, 의료 연구소 핵심 구역
- 존재량: 게임당 2-3개 제한

기술 매뉴얼 (극희귀)
- 효과: 기술/작업 관련 특성 1개 습득  
- 획득처: 첨단 연구소, 기업 R&D 센터
- 존재량: 게임당 2-3개 제한

지휘 교범 (전설급)
- 효과: 정신/지휘 관련 특성 1개 습득
- 획득처: 군 지휘부, 정부 핵심 시설
- 존재량: 게임당 1개 제한
```

#### 경험 누적을 통한 획득
```
장기간 특정 활동 반복:
- 전투 100회 승리 → 전투 관련 특성 후보 생성
- 건설 50회 완료 → 작업 관련 특성 후보 생성  
- 치료 30회 성공 → 의료 관련 특성 후보 생성
- 탐사 20회 완료 → 생존 관련 특성 후보 생성

특별한 경험:
- 보스 몬스터 단독 처치 → 특수 전투 특성
- 혁신적 기술 개발 → 특수 기술 특성
- 팀 전체 위기 극복 → 지휘 관련 특성
- 불치병 완치 → 특수 의료 특성

※ 경험 누적으로는 후보만 생성, 최종 습득은 플레이어 선택
```

## 📈 성장 메커니즘

### 능력치 성장 방식

#### 경험치 기반 성장
```
행동별 경험치 획득:
- 전투 승리: 전투, 사격 경험치
- 건설 완료: 작업, 기술 경험치  
- 치료 성공: 의료 경험치
- 탐사 완료: 체력, 민첩 경험치
- 위기 극복: 정신 경험치
```

#### 성장 한계와 속도
```
성장 구간별 필요 경험치:
- 1-10: 빠른 성장 (50경험치/레벨)
- 11-15: 보통 성장 (100경험치/레벨)
- 16-20: 느린 성장 (200경험치/레벨)
- 21-25: 매우 느린 성장 (500경험치/레벨)

개인별 재능 한계:
- 각 캐릭터마다 능력치별 최대치 존재
- 초기 생존자: 평균 15-18 한계
- 소환 캐릭터: 다양한 한계 (8-25)
```

## ⚖️ 능력치별 효과 상세

### 전투 능력치 효과
```
1-5 (비전투원): 전투 시 생존에만 집중, 공격력 최소
6-10 (초보자): 기본 무기 사용 가능, 단순 전투 참여
11-15 (숙련자): 효과적 전투 가능, 전술 이해
16-20 (전문가): 뛰어난 전투력, 어려운 적 상대 가능
21-25 (전설급): 압도적 전투력, 보스급 적 단독 상대
```

### 기술 능력치 효과  
```
1-5 (일반인): 단순 작업만 가능, 복잡한 장비 사용 불가
6-10 (초급자): 기본 도구 사용, 간단한 수리
11-15 (중급자): 전문 장비 조작, 고급 건설 참여
16-20 (전문가): 복잡한 시스템 설계, 연구 개발
21-25 (천재급): 혁신적 기술 개발, 차원 기술 연구
```

### 의료 능력치 효과
```
1-5 (응급처치): 기본 상처 치료, 간단한 의료 지원
6-10 (간병인): 환자 돌봄, 기본 의료 절차
11-15 (간호사): 전문 의료 지원, 중급 치료
16-20 (의사): 수술 가능, 복잡한 의료 시술
21-25 (명의): 기적적 치료, 불치병 연구
```

## 🎯 특성 조합 시너지

### 특성 조합 효과
```
근접 전문가 + 방어 전문가 = 탱커
- 추가 효과: 적 어그로 집중, 팀원 보호 +30%

사격 명수 + 정찰 전문가 = 저격수  
- 추가 효과: 원거리 크리티컬 +20%, 적 정보 수집

연구자 + 바이러스 전문가 = 백신 개발자
- 추가 효과: 백신 개발 속도 +100%, 감염 완전 치료

지휘관 + 협상가 = 완벽한 리더
- 추가 효과: 팀 전체 능력치 +2, 외교 성공률 +50%
```

## 🔄 육성 전략 가이드

### 초기 생존자 육성 방향
```
특성 없는 상태에서 시작:
- 경험 누적을 통한 자연스러운 특성 개발
- 희귀 아이템 획득시 전략적 특성 부여
- 개인별 역할에 맞는 특성 조합 구축

전투 특화 육성:
- 목표: 전투 18+, 사격 15+
- 특성 목표: 근접 전문가, 사격 명수
- 아이템 우선순위: 전투 교범
- 역할: 주력 전투원

기술 특화 육성:  
- 목표: 작업 20+, 기술 18+
- 특성 목표: 숙련 기술자, 연구자
- 아이템 우선순위: 기술 매뉴얼
- 역할: 거점 개발 핵심

의료 특화 육성:
- 목표: 의료 20+, 정신 15+
- 특성 목표: 외과 전문의, 멘탈 케어
- 아이템 우선순위: 의학 서적
- 역할: 팀 생존 담당
```

### 소환 캐릭터 활용법
```
태생 특성 보유 캐릭터:
- 즉시 전문 역할 배치 가능
- 기존 팀원과 특성 조합 고려
- 부족한 전문성 즉시 보완

고성능 + 다중 특성:
- 팀의 핵심 전력으로 활용
- 희귀 아이템으로 추가 특성 부여시 최대 효과
- 손실시 치명적이므로 안전 우선

저성능 캐릭터:
- 경험 누적을 통한 장기 육성
- 특성 아이템 투자 가치 신중 검토
- 안전한 후방 지원 역할부터 시작
```
