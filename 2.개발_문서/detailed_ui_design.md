## 🔝 상단 UI 시스템

### 상단 정보바 (1920x60)
```
┌─ 게임 정보 ──────────────────────────────────────────────┐
│ [🎮 폐교생존] 📅Day 3  🕐14:30  ⚡Wave 5  👥12/15명    │
│ 왼쪽: 게임 로고                  오른쪽: 시스템 버튼들   │
│                           [⏸️][⚙️][💾][❌]            │
└──────────────────────────────────────────────────────────┘

구성 요소:
- 게임 제목 (좌측 상단)
- 현재 날짜 및 시간
- 다음 웨이브 정보
- 생존자 수 (현재/최대)
- 시스템 버튼 (일시정지, 설정, 저장, 종료)
```

### 자원 상태 표시 (1920x80)
```
┌─ 자원 현황 ──────────────────────────────────────────────┐
│ 🍞45/50  💧78/100  ⛽12/30  🔧23/100  🏥8  📦15/20   │
│ ├─────┤  ├──────┤  ├────┤   ├──────┤  ├─┤  ├────┤     │
│ 식량     물        연료     스크랩    의약품 전자부품    │
│                                                        │
│ 🔔 최근 알림: "바리케이드 건설 완료"     [모든 알림] ▼   │
└──────────────────────────────────────────────────────────┘

특징:
- 아이콘 + 현재량/최대량
- 진행바로 시각적 표시
- 부족시 빨간색, 풍족시 녹색
- 클릭시 상세 정보 툴팁
- 우측에 최근 알림 1개 표시
```

## 🎮 메인 게임 뷰

### 카메라 및 선택 표시
```
카메라 컨트롤:
- WASD 또는 가장자리 스크롤로 이동
- 마우스 휠로 확대/축소
- 미니맵 클릭으로 즉시 이동

선택 표시 시스템:
┌─ 생존자 선택 ─────────────────────────────────┐
│     👤                                       │
│   ●───●  ← 녹색 원형 선택 표시               │
│                                              │
│ 다중 선택시:                                 │
│ 👤    👤    👤                              │
│●───● ●───● ●───●                            │
│녹색   파랑   노랑  ← 각기 다른 색상          │
└──────────────────────────────────────────────┘

상태 아이콘 (머리 위 표시):
- 🔧: 건설 작업 중
- ⚔️: 전투/경비 중  
- 🏥: 치료 중
- 😴: 휴식 중
- ⚠️: 위험 상태
- 📦: 운반 중
```

### 건설 모드 표시
```
그리드 오버레이:
┌─ 건설 그리드 ─────────────────────────────────┐
│ ┌─┬─┬─┬─┬─┬─┬─┬─┬─┬─┐                      │
│ ├─┼─┼─┼─┼─┼─┼─┼─┼─┼─┤                      │
│ ├─┼─┼█┼█┼─┼─┼─┼─┼─┼─┤ █: 기존 건물          │

│ ├─┼─┼─┼─┼░┼░┼─┼─┼─┼─┤ ▓: 선택된 위치        │
│ ├─┼─┼─┼─┼▓┼▓┼─┼─┼─┼─┤ ✗: 건설 불가          │
│ ├─┼─┼─┼─┼░┼░┼─┼─┼─┼─┤                      │

건설 미리보기:
- 반투명 건물 이미지
```

### 상황별 모드 전환
```
웨이브 방어 모드:
- 화면 가장자리에 빨간 테두리
- 상단에 적 카운터 표시
- 생존자 체력바 항상 표시
- 긴급 수리/치료 버튼 활성화

액션 모드:
- 조작 캐릭터 중앙 고정
- 하단에 체력/스태미나 바
- 스킬 쿨다운 표시
- 크로스헤어 또는 조준점

건설 모드:
- 그리드 오버레이 활성화
- 건설 메뉴 확장 표시
- 자원 요구사항 실시간 표시
- 배치 불가능 구역 하이라이트
```
