# 조작법 & 컨트롤 시스템

## 🎮 기본 조작 철학

### 핵심 설계 원칙
- **하이브리드 조작 시스템**: 워크래프트의 정밀성 + 롤의 현대적 편의성
- **스마트 우클릭**: 상황에 맞는 자동 명령 판단 + 수동 명령 옵션
- **스타크래프트 스타일 다중 선택**: 드래그 박스와 컨트롤 그룹 활용
- **적응형 인터페이스**: 초보자도 쉽게, 고수는 더 정밀하게

## 🖱️ 기본 조작법 (RTS 모드)

### 마우스 조작 (하이브리드 시스템)
```
좌클릭 (Left Click):
- 생존자 단일 선택
- UI 버튼 클릭
- 메뉴 선택
- 대상 지정 (명령 입력 후)

우클릭 (Right Click) - 스마트 캐스트:
- 빈 공간: 이동 명령
- 적: 공격 명령 (자동 판단)
- 자원: 수집 명령 (작업자인 경우)
- 아군: 추적/지원 명령
- 시설: 상호작용/수리 (상황에 따라)
- 부상자: 치료 명령 (의료진인 경우)

※ 스마트 캐스트 비활성화 가능 (설정에서)
※ 정밀 명령이 필요할 때는 전용 키 사용

드래그 (Click & Drag):
- 다중 선택 박스
- 카메라 이동 (중클릭 드래그)
- 스킬 방향 지정 (액션 모드)

휠 스크롤 (Wheel):
- 확대/축소 (Zoom In/Out)
- 메뉴 스크롤 (UI 위에서)
- 무기/도구 교체 (액션 모드)

중클릭 (Middle Click):
- 카메라 자유 이동 모드
- 미니맵 즉시 이동
- 빠른 정보 보기 (생존자/시설)
```

### 명령 시스템 (하이브리드)
```
기본 명령 (선택사항):
- A: 공격 명령 (Attack) - 강제 공격 모드
- S: 정지 명령 (Stop) - 즉시 실행
- H: 제자리 대기 (Hold Position) - 즉시 실행
- P: 순찰 모드 (Patrol) - 두 지점 클릭

작업 명령 (선택사항):
- G: 자원 수집 (Gather) - 강제 수집 모드
- R: 수리 명령 (Repair) - 강제 수리 모드
- L: 치료 명령 (heal/Life) - 강제 치료 모드
- F: 추적 명령 (Follow) - 다른 생존자 클릭

편의 기능 (현대적):
- 더블클릭: 같은 타입 모든 생존자 선택
- Ctrl + 클릭: 해당 생존자를 컨트롤 그룹에 추가
- Shift + 우클릭: 명령 대기열에 추가
- Alt + 우클릭: 공격 이동 (이동하면서 적 공격)

스마트 기능:
- 자동 타겟팅: 공격 중 적이 죽으면 근처 다른 적 자동 공격
- 자동 회피: 위험한 상황에서 자동으로 후퇴 (설정 가능)
- 자동 치료: 체력이 낮아지면 의료진에게 자동 요청 (설정 가능)
- 자동 수리: 내구도가 낮은 시설 자동 수리 알림

명령 실행 과정:
방법 1 (롤 스타일): 생존자 선택 → 우클릭 (스마트 판단)
방법 2 (워크래프트 스타일): 생존자 선택 → 명령 키 → 대상 클릭
→ 플레이어가 선호하는 방식 선택 가능!
```

## 👥 선택 및 그룹 관리 (스타크래프트 스타일)

### 선택 시스템
```
단일 선택:
- 좌클릭: 개별 생존자 선택
- 더블클릭: 같은 타입 모든 유닛 선택 (같은 역할 수행 중인 생존자들)

다중 선택:
- 드래그 박스: 여러 생존자 동시 선택
- Shift + 좌클릭: 선택 추가/제거
- Ctrl + 좌클릭: 같은 타입 모든 생존자 선택

선택 표시:
- 선택된 생존자: 발 밑에 녹색 원형 표시
- 다중 선택시: 각각 다른 색상 테두리
- 리더: 더 굵은 황금색 테두리
```

### 컨트롤 그룹 시스템
```
그룹 설정:
- Ctrl + 1~9: 현재 선택된 생존자들을 해당 번호에 저장
- Ctrl + Shift + 1~9: 기존 그룹에 추가

그룹 선택:
- 1~9: 해당 번호 그룹 선택
- 더블탭 (1~9 연속 두 번): 그룹 선택 + 카메라 이동

그룹 관리:
- Shift + 1~9: 해당 그룹에 현재 선택 추가
- Alt + 1~9: 해당 그룹에서 현재 선택 제거

예시 그룹 구성:
- 1: 전투팀 (리더 + 전투원들)
- 2: 작업팀 (기술자 + 건설 인력)
- 3: 의료팀 (의료진 + 보조원)
- 4: 탐사팀 (정찰 + 호위)
```

## 🏗️ 건설 모드 조작

### 건설 모드 진입
```
메뉴 방식:
- 하단 [건설] 버튼 클릭
- B키 단축키

그리드 표시:
- 건설 모드 진입시 10x10 그리드 자동 표시
- 건설 가능: 연한 녹색 하이라이트
- 건설 불가: 빨간색 X 표시
- 기존 건물: 회색 표시
```

### 건설 조작
```
시설 선택:
- 마우스로 건설 메뉴에서 시설 클릭
- 또는 단축키 (Q,W,E,R,T,Y... 순서)

배치 및 확정:
- 마우스 이동: 건설 미리보기 표시
- 좌클릭: 건설 위치 확정
- 우클릭: 건설 모드 취소
- R키: 회전 가능한 건물 회전
- Shift + 좌클릭: 연속 배치 모드

건설 큐 관리:
- 큐에 추가된 건설: 점선 테두리로 표시
- 건설 순서 변경: 드래그 앤 드롭
- Del키: 선택된 건설 계획 취소
```

## ⚔️ 액션 모드 조작

### 모드 전환
```
진입 방법:
- 스페이스바: RTS ↔ 액션 모드 토글
- 웨이브 시작시 자동 액션 모드 제안
- 위험 상황시 자동 전환 옵션

조작 대상 선택:
- Tab: 조작 가능한 생존자 순환
- F1~F7: 특정 생존자 즉시 선택
- 마우스 클릭으로도 전환 가능
```

### 액션 조작법
```
이동:
- WASD: 캐릭터 직접 이동
- 마우스: 시선 방향 조절

전투:
- 좌클릭: 공격 (근접무기/총기)
- 우클릭: 보조 액션 (회피, 블록, 조준)
- 스페이스: 특수 액션 (대쉬, 점프 등)
- R키: 재장전 (총기류)

상호작용:
- E키: 상호작용 (문 열기, 아이템 줍기, 스위치 조작)
- F키: 응급처치 (다른 생존자나 자신)
- G키: 수류탄 투척 (보유시)
- C키: 웅크리기 (은밀 이동)

아이템:
- 1~5: 퀵슬롯 아이템 사용
- I키: 인벤토리 열기
- 마우스 휠: 무기 교체
```

## 📹 카메라 시스템

### 카메라 조작
```
기본 이동:
- WASD: 부드러운 카메라 이동
- 화살표키: WASD와 동일 기능
- 마우스 가장자리: 화면 끝으로 마우스 이동시 자동 스크롤

확대/축소:
- 마우스 휠: 부드러운 줌 인/아웃
- +/-키: 단계별 확대/축소
- Home키: 기본 줌 레벨로 복귀

빠른 이동:
- 미니맵 클릭: 해당 위치로 즉시 이동
- 더블클릭: 해당 위치로 이동 + 약간 확대
- 스페이스: 최근 알림 위치로 이동
```

### 카메라 기능
```
고정 위치 저장:
- Ctrl + F1~F5: 현재 카메라 위치 저장
- F1~F5: 저장된 위치로 즉시 이동
- Shift + F1~F5: 저장된 위치로 이동 + 해당 위치 생존자 선택

자동 추적:
- 선택된 생존자 자동 추적 (옵션)
- 액션 모드시 조작 캐릭터 자동 추적
- 웨이브 공격시 위험 지역 자동 포커스
```

## 🎯 최적화된 워크플로우

### 일반적인 플레이 패턴 (하이브리드)
```
초보자 플레이 스타일 (롤 방식):
1. 생존자 선택 (클릭 또는 드래그)
2. 우클릭으로 모든 명령 (스마트 판단)
3. 자동화 옵션 활용
4. UI 버튼으로 기능 접근

고급자 플레이 스타일 (워크래프트 방식):
1. 컨트롤 그룹으로 빠른 선택 (1~9키)
2. 전용 키로 정밀한 명령 (A, G, R, L 등)
3. Shift로 복합 명령 설정
4. 단축키로 모든 기능 접근

자원 수집 (두 가지 방법):
방법 1: 작업자 선택 → 자원 우클릭 (자동 수집)
방법 2: 작업자 선택 → G키 → 자원 클릭 (강제 수집)

건설 (세 가지 방법):
방법 1: 작업자 우클릭 → 컨텍스트 메뉴 → 건설
방법 2: B키 → 건물 선택 → 위치 클릭
방법 3: 하단 UI 건설 버튼 → 건물 선택 → 위치 클릭

전투 (상황별 선택):
일반적: 전투팀 선택 → 적 우클릭 (자동 공격)
정밀한: 전투팀 선택 → A키 → 정확한 위치 클릭
연속적: Shift + 여러 적 우클릭 (순서대로 공격)

위기 상황 대응:
```
