# 자원 관리 시스템

## 📋 개요

### 자원과 아이템 구분
- **자원 (Resources)**: UI에 수치로 표시되어 게임 시스템 운영에 사용되는 재화
- **아이템 (Items)**: 인벤토리에 저장되어 플레이어의 직접적인 사용 또는 효과를 발휘하는 재화

## 🔧 자원 (Resources)

### 생존 필수 자원
| 자원명 | 특징 | 일일 소모량 | 획득처 | 사용처 |
|-------|------|------------|--------|--------|
| 식량 | 생존에 필수적인 자원 | 생존자 1명당 1단위/일 | 밭, 주방, 식료품점 탐사 | 생존 필수 자원 |
| 물 | 오염되지 않은 깨끗한 물 | 생존자 1명당 2단위/일 | 우물, 빗물통, 정수시설 | 생존 필수, 밭 운영, 차원 소환 |
| 연료 | 차량, 발전기 가동용 화석연료 | 발전기 1대당 3단위/일 | 고위험 지역, 주유소 | 차량, 시설 유지, 발전기, 차원 소환 |

### 제작 자원
| 자원명 | 특징 | 획득처 | 사용처 |
|-------|------|--------|--------|
| 스크랩 | 다양한 형태로 가공 가능한 금속성 자원 | 잔해, 폐차, 정제 및 가공 | 시설 제작, 장비 수리/제작, 차원 소환 |
| 목재 | 건축 및 가구 제작용 자원 | 주거지구 탐사, 가구 분해 | 기본 건설, 바리케이드, 휴게소 |
| 직물 | 의복 및 침구 제작용 자원 | 의류매장 탐사, 의류 분해 | 의복 제작, 휴게시설, 방한 대책 |

### 특수 자원
| 자원명 | 특징 | 획득처 | 사용처 |
|-------|------|--------|--------|
| 전자부품 | 고급 장비 제작용 부품 | 전자상가 탐사, 기기 분해 | 전기 시설, 차원 소환, 고급 장비 |
| 희귀광물 | 특수 아이템 제작용 광물 | 광산 지역 탐사 | 최고급 장비, 특수 연구 |

### 자원 저장 한계
- 식량: 50단위 (냉장고로 확장 가능)
- 물: 100단위
- 연료: 30단위 (인화성 위험)
- 스크랩: 100단위
- 목재: 100단위
- 직물: 60단위
- 전자부품: 50단위 (희귀 자원)
- 희귀광물: 20단위 (극희귀)

## 🎒 아이템 (Items)

### 의약품
- **특징**: 생존자의 건강 회복과 치료를 위한 아이템
- **획득처**: 약국, 제약회사, 병원, 거래
- **사용처**: 부상 치료, 체력 회복, 고가치 자원, 차원 소환

### 소모품
- **특징**: 소모 시 특수 효과를 획득하는 아이템
- **획득처**: 전 지역, 제작, 거래
- **사용처**: 특정 아이템 획득, 체력 회복, 버프 등

### 장비
- **특징**: 장착 가능한 아이템, 스텟 상승 효과
- **획득처**: 전 지역, 제작, 거래
- **사용처**: 장비 슬롯 장착 시 스텟 상승, 스킬 사용 가능

### 기타 아이템
- **특징**: 기타 잡템, 제작 재료, 스토리 관련 아이템
- **획득처**: 전 지역, 제작, 거래
- **사용처**: 제작, 거래

### 특수 아이템 (캐릭터 육성용)
- **경험치 포션**: 소량(50) / 중량(150) / 대량(400) / 전설(1000)
- **특성 교본**: 전투 교범, 의학 서적, 기술 매뉴얼, 지휘 교범
- **고급 전자부품**: 차원 소환 확률 2배 효과

## 🏭 자원 획득 시스템

### 수집 (탐사)
```
야외 탐색:
- 버려진 건물: 식량, 의약품, 전자부품
- 차량: 연료, 스크랩, 장비
- 자연 채집: 물, 목재, 희귀광물

지역별 특화 자원:
- 주거지구: 식량, 직물, 목재
- 상업지구: 식량, 의약품, 전자부품
- 공업지구: 스크랩, 연료
- 의료시설: 의약품, 의료용 아이템
- 연구시설: 전자부품, 특수 아이템
```

### 생산 (거점 내)
```
농업:
- 밭: 식량 생산 (주기적)
- 온실: 연중 생산 (전력 소모)

가공:
- 주방: 원재료를 식량으로 가공
- 작업대: 스크랩을 장비로 제작
- 연구실: 의약품 제작

분해:
- 가구 → 목재 3-5개
- 전자기기 → 전자부품 1-2개
- 차량 → 스크랩 10-15개 + 연료 5-10개
- 의류 → 직물 1-2개
```

### 거래
```
NPC 거래:
- 행상인과 물물교환
- 다른 생존자 집단과 거래
- 캐러밴 상단 방문

내부 교환 (작업대):
- 목재 3개 ↔ 스크랩 1개
- 전자부품 1개 ↔ 의약품 2개
- 직물 2개 ↔ 식량 3개
```

## 📦 저장 및 관리

### 저장 시설
```
창고 (자원 보관):
- 기본 용량: 각 자원별 기본 한계
- 일반 저장고: 모든 자원 +50% 확장
- 전용 저장고: 특정 자원 +100% 확장

냉장고 (식량 보존):
- 전력 소모: 1단위/일
- 효과: 식량 부패 완전 방지
- 용량: 식량 +200% 저장

특수 보관함:
- 위험물질 보관 (연료, 화약)
- 의약품 습도 조절
- 전자부품 EMP 차폐
```

### 운송 시스템
```
인력 운반:
- 운반량: 체력 × 2 단위
- 장거리시 피로도 증가

차량 운송:
- 운반량: 50-200 단위 (차종별)
- 연료 소모 필요
- 고장/조우 위험

수레/썰매:
- 운반량: 20-30 단위
- 연료 불필요, 조용
- 속도 느림
```

### 인벤토리
```
개인 소지품:
- 기본 10슬롯 (가방으로 확장)
- 체력에 따른 무게 제한

공용 창고:
- 자원: 자동 저장
- 아이템: 수동 입출고
```

## ⚖️ 자원 균형 설계

### 일일 소모량 (기본 7명 거점 기준)
```
필수 소모:
- 식량: 7단위/일
- 물: 14단위/일
- 연료: 3단위/일 (발전기 1대)

선택적 소모:
- 전자부품: 소환시 3개
- 의약품: 치료시 1-3개
- 건설 재료: 프로젝트별
```

### 단계별 자원 중점
```
초기 (1-5웨이브):
- 중점: 식량, 물, 목재
- 목표: 기본 생존 + 필수 건설

중기 (6-12웨이브):
- 중점: 스크랩, 전자부품, 의약품
- 목표: 고급 건설 + 소환 + 업그레이드

후기 (13웨이브+):
- 중점: 희귀광물, 특수 아이템
- 목표: 최고급 소환 + 연구 + 특수 건설
```

## 🚨 자원 부족 위기 관리

### 부족시 효과
```
식량 부족:
- 1일차: 경고
- 3일차: 작업 효율 -10%
- 7일차: 체력 감소, 사기 저하
- 14일차: 생존자 이탈 위험

물 부족:
- 1일차: 즉시 체력 감소
- 3일차: 질병 위험
- 7일차: 생존자 사망 위험

연료 부족:
- 즉시: 전력 시설 중단
- 야간: 보안 능력 저하
- 통신: 탐사팀 연락 두절
```

### 비상 대책
- 긴급 탐사 (위험 무시)
- 배급 제한 (소모량 50% 감소)
- 물물교환으로 필수 자원 확보
- 다른 생존자 집단에 구조 요청

## 📈 자원 관리 전략

### 초보자 가이드
1. 식량과 물 확보 (최우선)
2. 저장고 건설 (용량 확장)
3. 발전기 설치 (전력 확보)
4. 작업대 건설 (자급자족)

### 고급 전략
- 생산 특성 생존자 우선 배치
- 시설 업그레이드로 생산량 증대
- 자원별 전문 저장고 건설
- 최소 7일분 비상 자원 보유
- 다양한 획득처 확보로 위험 분산
