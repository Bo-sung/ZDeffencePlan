# 전투/방어 시스템

## ⚔️ 웨이브 방어 메커니즘

### 웨이브 주기 시스템
```
웨이브 간격: 게임 내 3-5일 (실제 시간 1-1.5시간)
- 초기 웨이브 (1-3): 3일 간격
- 중기 웨이브 (4-8): 4일 간격  
- 후기 웨이브 (9+): 5일 간격

웨이브 예고 시간: 2-6시간 전 사전 경고
- 정찰 정보: 적 규모 및 유형 예상
- 준비 시간: 방어 배치 및 긴급 건설
- 긴장감 조성: 카운트다운 타이머
```

### 웨이브 규모 증가
```
웨이브별 적 수량:
- 웨이브 1-2: 5-8마리 (튜토리얼 수준)
- 웨이브 3-5: 12-18마리 (기본 난이도)
- 웨이브 6-8: 20-30마리 (중간 난이도)
- 웨이브 9-12: 35-50마리 (높은 난이도)
- 웨이브 13+: 60마리+ (극한 난이도)

적 구성 변화:
- 초기: 90% 일반 몬스터 + 10% 중급
- 중기: 70% 일반 + 25% 중급 + 5% 고급
- 후기: 50% 일반 + 35% 중급 + 15% 고급
```

## 👹 몬스터 유형 시스템

### 일반 몬스터 (기본 위협)
```
감염된 인간
- 체력: 80HP
- 이동속도: 보통
- 공격력: 15
- 특성: 바이러스 전파 (접촉시 감염 위험 10%)
- 약점: 머리 공격시 2배 피해

좀비견
- 체력: 60HP  
- 이동속도: 빠름
- 공격력: 12
- 특성: 빠른 이동, 소형 목표
- 약점: 낮은 체력, 원거리 공격에 취약

돌진 감염자
- 체력: 120HP
- 이동속도: 매우 빠름 (돌진시)
- 공격력: 25 (돌진 공격)
- 특성: 직선 돌진으로 바리케이드 파괴
- 약점: 돌진 후 3초간 기절
```

### 중급 몬스터 (전술적 위협)
```
스피터 (원거리 공격)
- 체력: 100HP
- 사거리: 6타일
- 공격력: 20 (독성 투사체)
- 특성: 후방에서 지속 공격, 독 데미지
- 약점: 근접 공격에 취약

크롤러 (은신 공격)
- 체력: 90HP
- 이동속도: 느림 → 매우 빠름 (발견시)
- 공격력: 30 (기습 공격)
- 특성: 방어선 우회, 후방 침투
- 약점: 감시탑으로 사전 발견 가능

부풀어오른 감염자 (자폭)
- 체력: 150HP
- 이동속도: 느림
- 폭발 피해: 80 (반경 2타일)
- 특성: 사망시 자동 폭발, 바리케이드 대량 파괴
- 약점: 원거리에서 제거 필수
```

### 고급 몬스터 (보스급 위협)
```
감염된 거대 곰 (탱커형)
- 체력: 500HP
- 이동속도: 느림
- 공격력: 50
- 특성: 바리케이드 무시, 높은 방어력
- 약점: 이동 속도 느림, 함정에 취약

차원 촉수 (원거리 보스)
- 체력: 400HP
- 사거리: 8타일
- 공격력: 40 (다중 타겟)
- 특성: 공중 공격 (바리케이드 무시)
- 약점: 감시탑에서만 공격 가능

감염된 기계 (하이브리드)
- 체력: 350HP
- 이동속도: 보통
- 공격력: 35
- 특성: EMP 공격 (전자 시설 무력화 3분)
- 약점: 전자부품 공격으로 추가 피해
```

### 특수 보스 (웨이브 10+ 등장)
```
차원 지배자 (웨이브 15, 20, 25...)
- 체력: 1000HP
- 다중 공격 패턴
- 특수 능력: 몬스터 소환, 시설 파괴, 바이러스 폭발
- 처치 보상: 전설급 아이템, 대량 자원

감염된 탱크 (웨이브 12, 18, 24...)
- 체력: 800HP
- 포격 공격: 원거리 대규모 피해
- 특성: 극도로 높은 방어력
- 약점: 측면/후면 공격시 추가 피해
```

## 🛡️ 방어 메커니즘

### 방어선 구성
```
1차 방어선: 담벼락 + 바리케이드
- 적 이동속도 50% 감소
- 기본 내구도: 200HP (바리케이드당)
- 수리 가능: 목재 2개 + 작업시간 1시간

2차 방어선: 중앙 운동장 시설
- 감시탑: 원거리 공격 + 시야 확보
- 함정: 이동 방해 + 피해
- 발전기: 전력 공급 (조명, 전자 시설)

최후 방어선: 학교 건물 내부
- 복도 바리케이드: 좁은 통로 활용
- 교실 거점: 생존자 최후 저항 지점
- 비상 탈출로: 체육관 차원 균열 활용
```

### 생존자 전투 배치
```
원거리 배치 (감시탑, 옥상):
- 사격 능력치 15+ 권장
- 안전하지만 제한적 화력
- 탄약 소모량 고려

근거리 배치 (바리케이드 라인):
- 전투 능력치 12+ 필수
- 높은 위험, 높은 효과
- 부상 위험 상존

유동 배치 (기동 대응팀):
- 민첩 능력치 14+ 권장
- 돌파구 차단, 긴급 수리
- 전술적 유연성 제공
```

## ⚔️ 전투 계산 시스템

### 피해 계산 공식
```
기본 피해 = (무기 공격력 + 능력치 보너스) × 거리 보정 × 특성 보정

능력치 보너스:
- 근접 전투: 전투 능력치 × 0.5
- 원거리 전투: 사격 능력치 × 0.5

거리 보정:
- 근접 (1타일): 100%
- 중거리 (2-4타일): 80%
- 원거리 (5-8타일): 60%

특성 보정:
- 근접 전문가: 근접 공격 +25%
- 사격 명수: 원거리 공격 +30%, 크리티컬 +10%
- 방어 전문가: 받는 피해 -20%
```

### 크리티컬 시스템
```
기본 크리티컬 확률: 5%
+ 사격 능력치 15+: +3%
+ 사격 명수 특성: +10%
+ 야간 작전 특성: 야간 +15%

크리티컬 효과:
- 피해 2배 적용
- 몬스터 기절 효과 (1-2초)
- 특수 몬스터 약점 공격시 3배
```

## 🩹 부상 및 사망 시스템

### 전투 중 부상 확률
```
안전 배치 (감시탑, 후방):
- 부상 확률: 웨이브당 5%
- 주로 경상 (유탄 파편, 낙하)

위험 배치 (바리케이드, 전방):
- 부상 확률: 웨이브당 20%
- 경상~중상 가능

매우 위험 (바리케이드 파괴 후):
- 부상 확률: 50%+
- 중상~사망 위험

부상 종류별 치료 시간:
- 경상: 2-4시간 (의료 10+ 필요)
- 중상: 8-16시간 (의료 15+ 필요)
- 중태: 24-48시간 (의료 20+ 필요)
```

### 사망 처리
```
사망 조건:
- 체력 0 도달 + 치료 실패
- 즉사 공격 (거대 몬스터 직격)
- 바이러스 감염 + 치료 실패

사망시 손실:
- 해당 캐릭터 완전 소실
- 착용 장비 50% 확률로 파괴
- 축적된 경험치 및 특성 완전 소실
- 팀 사기 -20% (3일간)

사망 방지 시스템:
- 의료진 즉시 투입
- 응급 치료 키트 사용
- 부상자 후방 이송
```

## 🎯 전투 전략 요소

### 웨이브별 대응 전략

#### 초기 웨이브 (1-3)
```
목표: 기본 방어선 구축 및 전투 경험 축적
추천 배치:
- 감시탑 1개 + 바리케이드 3개
- 원거리 2명 + 근거리 2명
- 의료진 1명 대기

핵심 전술:
- 안전한 원거리 공격 위주
- 부상 즉시 치료로 경험 축적
- 자원 소모 최소화
```

#### 중기 웨이브 (4-8)
```
목표: 효율적 방어선 운영 및 고급 몬스터 대응
추천 배치:
- 감시탑 2개 + 함정 5개 + 바리케이드 6개
- 원거리 3명 + 근거리 3명 + 기동팀 2명
- 의료진 1명 + 수리팀 1명

핵심 전술:
- 함정을 활용한 몬스터 분산
- 특수 몬스터 우선 타겟
- 바리케이드 실시간 수리
```

#### 후기 웨이브 (9+)
```
목표: 대규모 공격 생존 및 보스 몬스터 격퇴
추천 배치:
- 감시탑 3개 + 함정 10개 + 바리케이드 8개
- 전투 특화 인력 총동원
- 다층 방어선 구축

핵심 전술:
- 보스 몬스터 집중 공격
- 비상 계획 (내부 후퇴)
- 차원 소환 비상 병력 준비
```

### 특수 상황 대응

#### 바리케이드 돌파시
```
즉시 대응:
1. 기동팀 긴급 투입
2. 돌파구 임시 봉쇄
3. 생존자 후방 이송

추가 대책:
- 복도 바리케이드 활용
- 교실별 농성 준비
- 차원 균열 탈출 계획
```

#### 전력 시설 파괴시
```
영향:
- 감시탑 기능 정지
- 야간 시야 제한
- 통신 두절

대응책:
- 비상 발전기 가동
- 수동 조명 장비 활용
- 근거리 전투 강화
```

## 🏆 전투 보상 시스템

### 웨이브 클리어 보상
```
기본 보상:
- 경험치: 참여자 전원 +15
- 자원: 적 수량 × 0.5개 스크랩

특수 보상 (확률):
- 무기/방어구 (10% 확률)
- 의료용품 (15% 확률)
- 희귀 자원 (5% 확률)

완벽 방어 보너스 (피해 10% 미만):
- 추가 경험치 +10
- 특수 아이템 확률 2배
- 생존자 사기 +20%
```

### 몬스터별 드랍 아이템
```
일반 몬스터:
- 기본 자원 (스크랩, 화약)
- 바이러스 샘플 (연구용)

중급 몬스터:
- 중급 자원 (전자부품, 의약품)
- 특수 재료 (강화 소재)

고급 몬스터:
- 희귀 자원 (희귀광물)
- 고급 장비 재료

보스 몬스터:
- 전설급 아이템
- 대량 자원
- 특성 교본 (희귀)
```

## 📊 전투 밸런스 조정

### 난이도 조절 시스템
```
동적 난이도 조정:
- 연속 실패시: 적 수량 -10%
- 연속 성공시: 적 수량 +10%
- 플레이어 레벨 고려: 고급 생존자 비율에 따른 조정

플레이어 선택 난이도:
- 쉬움: 적 수량 -25%, 드랍률 +50%
- 보통: 기본 설정
- 어려움: 적 수량 +25%, 드랍률 -25%
- 지옥: 적 수량 +50%, 특수 몬스터 증가
```

### 밸런스 검증 요소
```
생존률 목표:
- 초보자: 웨이브 5까지 80% 생존
- 숙련자: 웨이브 10까지 60% 생존
- 전문가: 웨이브 15까지 40% 생존

자원 소모 균형:
- 웨이브당 평균 자원 소모 < 탐사 획득량
- 긴급 상황 대비 비축분 고려
- 장기전 지속 가능성 확보
```
